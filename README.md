# Bajaj Finserv Health - Java Qualifier

## Overview

Spring Boot app that:

* Generates a webhook on startup
* Solves a SQL problem
* Submits the solution using JWT authentication

## Tech Stack

* Java
* Spring Boot
* REST APIs

## How to Run

```
./mvnw.cmd spring-boot:run
```

## Workflow

1. Generate webhook
2. Receive token
3. Solve SQL query
4. Submit solution
